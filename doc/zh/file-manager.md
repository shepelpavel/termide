# 文件管理器

文件管理器面板提供直观的界面，用于浏览文件系统和执行文件及目录操作。

远程文件系统（SFTP / FTP / FTPS）与本地路径在同一面板中显示——
URL 语法和认证配置请参见[远程文件系统](vfs.md)。

## 导航

| 快捷键 | 操作 |
|-------------------|--------------------------------------------|
| `↑` / `↓`         | 上下移动光标                               |
| `Enter`           | 进入目录、预览媒体或打开文件               |
| `Backspace`       | 转到上级目录                               |
| `~`               | 转到主目录                                 |
| `PageUp` / `PageDown` | 按页滚动列表                          |
| `Home` / `End`    | 转到列表开头/末尾                          |
| `.`               | 切换隐藏文件可见性                         |
| `→` / `l`       | 展开目录（树形视图）                        |
| `←` / `h`       | 折叠目录（树形视图）                        |
| `/`              | 树内增量搜索                                |
| `Ctrl+/`          | 打开目录切换器                             |
| `Ctrl+G`          | 转到路径/URL                               |
| `Alt+K`           | 添加书签                                   |
| `Tab`             | 转到下一个面板                             |
| `Shift+Tab`       | 转到上一个面板                             |

## 文件选择

| 快捷键 | 操作 |
|-------------------|--------------------------------------------|
| `Insert`          | 切换当前文件的选择状态                     |
| `Shift + ↑ / ↓`   | 选择多个连续文件                           |
| `Ctrl+A`          | 选择面板中的所有文件和目录                 |
| `Escape`          | 清除所有选择                               |

在树形视图中，使用 `Insert` 选择目录会级联选择其中的所有文件。折叠已选择的目录时，其子元素的选择状态会保留。

| 快捷键 | 操作 |

| `Ctrl+N`          | 创建新文件                                 |
| `D` / `F7`        | 创建新目录                                 |
| `Delete` / `F8`   | 删除选中的文件/目录                        |
| `C` / `F5`        | 复制选中的文件/目录                        |

## 搜索

| 快捷键 | 操作 |
|-------------------|--------------------------------------------|
| `Ctrl+F`          | 按名称搜索文件（glob 模式）               |
| `Ctrl+Shift+F`    | 在文件内容中搜索                           |
| `Ctrl+Shift+H`    | 在文件内容中搜索并替换                     |
| `/`              | 树内增量搜索（边输入边过滤）                |

这些搜索都使用**停靠在面板顶部的内嵌栏**（而非浮动模态框），结果上方有分隔线。
栏与结果是两个**区域**：`Tab` 在它们之间切换（与 Git 状态面板一致）。在栏区域，
方向键在字段与开关间移动。在结果区域，光标落在**条目行**（文件/文件夹，或文件
标题）上，与 diff 面板一致：`↑` / `↓` 在它们之间移动，`PageUp` / `PageDown`
翻页，`←` / `→` 折叠 / 展开光标处的条目，`Enter` 打开它（文件夹/标题则折叠或
打开）。鼠标单击选择行，双击打开（或折叠分组），滚轮滚动。`Esc` 退出搜索
（不会关闭面板）。

### 文件搜索 (Ctrl+F)

内嵌栏，仅含一个 `Find:` 字段，实时按 glob 过滤文件；结果（带 Git 状态颜色的
相对路径）以树形显示在分隔线下方。`Tab` 进入结果；`↑` / `↓` 在文件**和文件夹**
间移动，`←` / `→` 折叠 / 展开文件夹，`Enter` 打开文件（并在树中将光标置于其上）。

### 内容搜索 (Ctrl+Shift+F)

内嵌栏，含 `Find:`（glob 掩码，默认 `*`）与 `Text:`（内容查询）字段：
- 默认按**字面**匹配；`[.*] Regex` 切换为正则表达式，`[Aa] Case` 切换大小写
  敏感（点击，或聚焦开关后按 `Enter` / `Space`）。
- 仅搜索文本文件（跳过二进制文件）；跳过大文件（可在设置中配置限制）。
- 结果在分隔线下方**按文件分组**，与 diff 面板一致：光标在**文件标题**间移动
  （`[▼]`/`[▶]` 折叠标记 + 路径 + 匹配数），每个文件下最多显示 5 行匹配（行号 +
  高亮匹配行），超出则显示 `+ N more` 行。`←` / `→` 折叠 / 展开文件，`Enter`
  在其首个匹配处打开。

### 内容替换 (Ctrl+Shift+H)

`Ctrl+Shift+H` 打开同一内容栏，并多出一个 `Repl:` 字段。输入替换文本后，每个
显示的匹配都会渲染为 `-旧/+新` 预览。

每个文件标题都有一个**选择复选框**（折叠三角右侧的 `[ ]`），替换**只作用于已
勾选的文件**——默认不勾选任何文件。用 `Space` 或点击切换光标处文件；**`[ ] Select all`** 复选框（或 `a`）
一次性全选/全不选。栏右侧实时显示
`已选 / 总文件 · 替换数` 计数。

在 `Repl:` 字段按 `Enter`（或激活 **Replace** 按钮）即可替换**已勾选**文件中的
全部匹配——会先确认替换的处数与文件数（未选择时显示提示）。开启 `[.*] Regex`
时替换支持 `$1` / `${name}` 捕获组；否则按字面插入。替换会写入磁盘。

### 树内搜索 (/)

按 `/` 在当前目录树中进行增量搜索：
- 输入时实时过滤文件列表，仅显示匹配项
- 匹配文件的父目录保持可见以提供上下文
- 按 `Enter` 确认并导航到匹配项
- 按 `Escape` 取消并恢复完整树
- 与树形视图协同工作 — 包含匹配项的目录自动展开

## 剪贴板

| 快捷键 | 操作 |
|-------------------|--------------------------------------------|
| `Ctrl+C`          | 复制选中项的路径                           |
| `Ctrl+X`          | 剪切选中项的路径                           |
| `Ctrl+V`          | 从剪贴板粘贴文件                           |

### 目标目录规则

新建文件、新建目录和从剪贴板粘贴都会**落在光标所在的树层级**，并非
总是落在面板根目录：

- 光标在顶层条目上 → 操作目标为面板的当前目录。
- 光标在展开的子目录内 → 操作目标为该子目录。

这符合光标的视觉位置——所见即所建/所贴位置。

## Git 集成

文件管理器显示 Git 仓库中的文件状态：

- **文件状态颜色** — 新增、修改、删除和未跟踪的文件以不同颜色标识
- **嵌套 Git 状态** — 目录显示其子元素的聚合状态（例如，包含修改文件的目录会被高亮显示）
- **树形视图集成** — Git 状态在目录树中传播，便于在深层层次结构中定位更改

## 媒体预览

文件管理器可以使用控制台图片查看器预览图片和视频。

**文件打开逻辑：**

| 文件类型 | 操作 |
|-----------|--------|
| 光栅图片（PNG、JPG、JPEG、GIF、WebP、BMP、TIFF） | ImagePanel（原生图形）或 xdg-open 回退 |
| 矢量图片（SVG、ICO） | xdg-open（系统查看器） |
| 视频（MP4、MKV、AVI、MOV、WebM、FLV、WMV、M4V） | xdg-open（系统播放器） |
| 二进制文件 | xdg-open（系统默认） |
| 文本文件 | 编辑器面板 |
| 可执行文件 | 在终端中运行 |

**快捷键：**
- `Enter` → 智能打开（参见上表）
- `F3` → 查看文件（类似 Enter，但可执行文件在编辑器中打开而非运行）
- `O` / `Alt+Enter` → 强制使用 xdg-open（系统默认应用程序）打开
- `F4` → 始终在编辑器中打开

**原生图形：**
termide 自动检测父终端是否支持图形协议（Kitty、Sixel、iTerm2）。支持时，光栅图片直接在 ImagePanel 中渲染，无需外部工具。

**支持的终端：**
- Kitty、WezTerm、iTerm2、Ghostty、foot - 完整图形支持
- 其他终端 - 回退到 xdg-open

## 鼠标支持

- **单击**：选择文件或目录
- **双击**：进入目录或打开文件
- **滚轮**：滚动文件列表

## 显示

### 字体修饰

- **斜体** — 符号链接（文件和目录）
- **粗体** — 可执行文件

### 排序顺序

文件按分组显示，每组按字母顺序排列：
1. 目录
2. 可执行文件
3. 普通文件

### 符号链接

- 复制模态框包含"创建符号链接"复选框 — 创建符号链接而非复制
- 进入符号链接目录时，文件管理器跟随链接进入目标目录
- 在符号链接上按 `Space` 打开属性模态框，其中包含 **Target** 行，显示链接中实际写入的路径（未经 `canonicalize` 解析），并将用户主目录缩写为 `~`
- 状态栏会在当前选中符号链接的文件名后显示 `→ target`；该行为同样适用于展开子目录树视图中的嵌套符号链接

### 备注

- `..`（上级目录）不可选择 — 仅用于导航
- 输入模态框支持将 `~/` 展开为主目录

## 配置

文件管理器从 `config.toml` 中的 `[file_manager]` 节读取设置。主要的
用户可见选项：

| 键                                  | 默认值   | 说明                                                                                                          |
|-------------------------------------|----------|---------------------------------------------------------------------------------------------------------------|
| `extended_view_width`               | `50`     | 面板宽度（列数）达到该值时 extended view 会显示大小/修改时间列。                                              |
| `content_search_max_file_size_mb`   | `1`      | `Ctrl+Shift+F` 内容搜索考虑的最大文件大小，超过则跳过。                                                       |
| `dir_size_in_wide_view`             | `true`   | 在 extended view 的 Size 列计算并显示目录大小。仅适用于本地文件系统。                                         |
| `dir_size_budget_ms`                | `100`    | 单个目录扫描的时间预算（毫秒）。超过预算的目录显示 `-`。设为 `0` 则关闭该功能。                                |

扫描结果保存在进程级共享缓存中，两个查看同一目录的面板不会重复扫描。
在目录上按 `Space` 也会把它的精确（完整）大小发布到该缓存——下一帧
重绘时 wide view 的大小列会读取该结果。
