# 文件管理器

文件管理器面板提供直观的界面，用于浏览文件系统和执行文件及目录操作。

## 导航

| 快捷键 | 操作 |
|-------------------|--------------------------------------------|
| `↑` / `↓`         | 上下移动光标                               |
| `Enter`           | 进入目录、预览媒体或打开文件               |
| `Backspace`       | 转到上级目录                               |
| `~`               | 转到主目录                                 |
| `PageUp` / `PageDown` | 按页滚动列表                          |
| `Home` / `End`    | 转到列表开头/末尾                          |
| `.`               | 切换隐藏文件可见性                         |
| `Ctrl+P`          | 打开目录切换器                             |
| `Ctrl+G`          | 转到路径/URL                               |
| `Ctrl+B`          | 添加书签                                   |
| `Tab`             | 转到下一个面板                             |
| `Shift+Tab`       | 转到上一个面板                             |

## 文件选择

| 快捷键 | 操作 |
|-------------------|--------------------------------------------|
| `Insert`          | 切换当前文件的选择状态                     |
| `Shift + ↑ / ↓`   | 选择多个连续文件                           |
| `Ctrl+A`          | 选择面板中的所有文件和目录                 |
| `Escape`          | 清除所有选择                               |

## 文件操作

| 快捷键 | 操作 |
|-------------------|--------------------------------------------|
| `Ctrl+N`          | 创建新文件                                 |
| `D` / `F7`        | 创建新目录                                 |
| `Delete` / `F8`   | 删除选中的文件/目录                        |
| `C` / `F5`        | 复制选中的文件/目录                        |
| `M` / `F6`        | 移动/重命名文件/目录                       |
| `E` / `F4`        | 在编辑器中打开文件                         |
| `R` / `F2`        | 重命名文件/目录                            |
| `V` / `F3`        | 查看文件（预览而不执行）                   |
| `Ctrl+R`          | 刷新当前目录内容                           |
| `Space`           | 显示文件/目录信息                          |

## 搜索

| 快捷键 | 操作 |
|-------------------|--------------------------------------------|
| `Ctrl+F`          | 按名称搜索文件（glob 模式）               |
| `Ctrl+Shift+F`    | 在文件内容中搜索（正则表达式）             |

### 文件搜索 (Ctrl+F)

打开模态框，使用 glob 模式快速按名称搜索文件：
- 输入文字即可实时过滤文件
- 结果显示相对路径，带有 Git 状态颜色
- 按 Enter 打开选中的文件
- 按 Escape 或点击外部区域关闭

### 内容搜索 (Ctrl+Shift+F)

打开模态框，使用正则表达式在文件内搜索文本：
- 仅搜索文本文件（跳过二进制文件）
- 跳过大文件（可在设置中配置限制）
- 结果显示文件路径、行号和上下文（3 行）
- 匹配的文本会被高亮显示
- 按 Enter 在匹配行处打开文件
- 按 Escape 或点击外部区域关闭

## 剪贴板

| 快捷键 | 操作 |
|-------------------|--------------------------------------------|
| `Ctrl+C`          | 复制选中项的路径                           |
| `Ctrl+X`          | 剪切选中项的路径                           |
| `Ctrl+V`          | 从剪贴板粘贴文件                           |

## Git 集成

文件管理器在 Git 仓库中显示文件状态，高亮显示新增、修改和删除的文件。

## 媒体预览

文件管理器可以使用控制台图片查看器预览图片和视频。

**文件打开逻辑：**

| 文件类型 | 操作 |
|-----------|--------|
| 光栅图片（PNG、JPG、JPEG、GIF、WebP、BMP、TIFF） | ImagePanel（原生图形）或 xdg-open 回退 |
| 矢量图片（SVG、ICO） | xdg-open（系统查看器） |
| 视频（MP4、MKV、AVI、MOV、WebM、FLV、WMV、M4V） | xdg-open（系统播放器） |
| 二进制文件 | xdg-open（系统默认） |
| 文本文件 | 编辑器面板 |
| 可执行文件 | 在终端中运行 |

**快捷键：**
- `Enter` → 智能打开（参见上表）
- `F3` → 查看文件（类似 Enter，但可执行文件在编辑器中打开而非运行）
- `Shift+Enter` → 强制使用 xdg-open（系统默认应用程序）打开
- `F4` → 始终在编辑器中打开

**原生图形：**
termide 自动检测父终端是否支持图形协议（Kitty、Sixel、iTerm2）。支持时，光栅图片直接在 ImagePanel 中渲染，无需外部工具。

**支持的终端：**
- Kitty、WezTerm、iTerm2、Ghostty、foot - 完整图形支持
- 其他终端 - 回退到 xdg-open

## 鼠标支持

- **单击**：选择文件或目录
- **双击**：进入目录或打开文件
- **滚轮**：滚动文件列表

## 显示

### 字体修饰

- **斜体** — 符号链接（文件和目录）
- **粗体** — 可执行文件

### 排序顺序

文件按分组显示，每组按字母顺序排列：
1. 目录
2. 可执行文件
3. 普通文件

### 符号链接

- 复制模态框包含"创建符号链接"复选框 — 创建符号链接而非复制
- 进入符号链接目录时，文件管理器跟随链接进入目标目录

### 备注

- `..`（上级目录）不可选择 — 仅用于导航
- 输入模态框支持将 `~/` 展开为主目录
