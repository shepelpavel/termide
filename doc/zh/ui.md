# 应用程序窗口概览

**应用程序占据所有可用空间，垂直划分为以下元素：**
- 菜单栏
- 面板区域
- 状态栏

应用程序还使用弹出窗口：
- 帮助
- 设置
- 关闭应用确认

## 模态窗口

应用程序使用交互式模态窗口进行各种操作：
- **搜索模态框**（`Ctrl+F`）- 具有实时预览、匹配计数和导航按钮的交互式搜索
- **替换模态框**（`Ctrl+H`）- 具有两个输入框、实时搜索和操作按钮的交互式替换
- **输入模态框** - 用于文件操作（创建、重命名等）的各种提示
- **确认对话框** - 删除确认、未保存更改等

**模态框功能：**
- 模态框标题栏中的 `[X]` 关闭按钮（可用鼠标点击）
- 使用 Tab/Shift+Tab 进行键盘导航
- 所有按钮支持鼠标操作
- Escape 键关闭模态框
- 搜索/替换操作的实时预览
- 状态保持（保存上次输入的文本）

## 菜单栏

菜单栏位于窗口顶部，包括：左侧的菜单项、系统资源指示器（CPU、内存、网络速度），以及右侧的 HH:MM 格式时钟。
菜单的激活/停用及各项目可通过鼠标点击或[键盘快捷键](#键盘导航和面板管理)访问。

**菜单项：**
- `Sessions` — 会话管理子菜单：
  - 新建会话 — 在新目录中创建会话
  - 切换会话 — 打开会话切换模态窗口
  - 更改根路径 — 将当前会话移至其他目录
- `Windows` — 面板创建子菜单：
  - Files — 文件管理器面板
  - Terminal — 终端面板（有子菜单可选择 shell：列出系统中所有可用的 shell，默认 shell 以 ● 标记）
  - Editor — 文本编辑器面板
  - Git Status — Git 状态面板
  - Git Log — 提交历史面板
  - Journal — 应用日志面板
  - Diagnostics — LSP 诊断面板
  - Operations — 后台操作面板
  - Outline — 代码结构导航面板
- `Scripts` — 用户自定义脚本（支持分组子菜单）
- `Bookmarks` — 书签（目录、文件、SSH、SFTP）
- `Options` — 设置子菜单：
  - Themes — 主题选择（实时预览）
  - Language — 界面语言（实时预览）
  - Manage scripts — 打开脚本文件夹
  - Manage bookmarks — 打开书签文件
  - Edit preferences — 在编辑器中打开 config.toml
  - Help — 帮助面板
  - Quit — 退出应用

**系统资源指示器：**
- `CPU` - CPU 使用率百分比，带颜色编码（绿色 < 50%，黄色 50-75%，红色 > 75%）
- `RAM` - 内存使用量（GB/MB 格式），根据使用水平进行颜色编码
- `↓/↑` - 网络下载/上传速度

## 面板区域

面板区域填充菜单栏和状态栏之间的垂直空间，从左到右延伸至窗口边缘。
默认情况下，该区域显示双面板文件管理器（类似 Midnight Commander）。其他面板（终端、编辑器、Git）可以打开并排列在旁边。

**可打开的面板类型：**
- [文件管理器](file-manager.md) — `Alt+F`
- [终端](terminal.md) — `Alt+T`
- [文本编辑器](editor.md) — `Alt+E`
- Git 状态 — `Alt+G`
- 大纲 — `Alt+O`
- 诊断 — `Alt+I`
- Git 日志 — `Alt+C`
- Git 差异
- 操作
- 图片查看器
- 帮助 — `Alt+H`
- 日志 — `Alt+L`

**可关闭面板的特点：**
- 面板标题中有 `[X]` 关闭按钮（可用鼠标点击）
- 可通过 Escape、Alt+X 或 Alt+Backspace 关闭
- 可通过 Alt+加号/减号调整大小

## 状态栏

状态栏用于显示活动面板的附加工作信息。
根据活动面板的类型，显示相应的数据。

### 磁盘空间指示器

状态栏在右侧显示磁盘空间信息，格式为：`设备名 已用/总计GB (使用率%)`，根据使用水平进行颜色编码：

**颜色编码：**
- **绿色** 磁盘使用率 < 50%
- **黄色** 磁盘使用率 50-75%
- **红色** 磁盘使用率 > 75%

**格式：** `设备名 已用/总计 (使用率%)`

示例：`NVME0N1P2 386/467Gb (83%)`

设备名称从文件系统自动检测：
- Linux 上：显示分区名称，如 `NVME0N1P2`、`SDA1` 等
- macOS 上：显示磁盘标识符
- 显示的设备对应当前目录所在的分区

## 键盘导航和面板管理

| 快捷键 | 操作 |
|-------------------|--------------------------------------------|
| `Alt+M`           | 激活/停用菜单                              |
| `Alt+F`           | 打开文件管理器面板                         |
| `Alt+T`           | 打开终端面板                               |
| `Alt+E`           | 打开新文件编辑器面板                       |
| `Alt+G`           | 打开 Git 状态面板                          |
| `Alt+O`           | 打开大纲面板                               |
| `Alt+I`           | 打开诊断面板                               |
| `Alt+C`           | 打开 Git 日志面板                          |
| `Alt+L`           | 打开日志查看器面板                               |
| `Alt+P`           | 在编辑器中打开配置文件                     |
| `Alt+H`           | 打开帮助窗口                               |
| `Alt+Q`           | 关闭应用程序                               |
| `Escape`          | 关闭面板 / 关闭模态框                      |
| `Alt+X`           | 关闭面板                                   |
| `Alt+Delete`      | 关闭面板                                   |
| `Alt+Left`        | 转到上一个面板组（水平方向）               |
| `Alt+Right`       | 转到下一个面板组（水平方向）               |
| `Alt+Up`          | 转到组内上一个面板（垂直方向）             |
| `Alt+Down`        | 转到组内下一个面板（垂直方向）             |
| `Alt+W/S/A/D`     | WASD 风格面板导航（箭头键的替代方案）      |
| `Alt+PgUp`        | 将面板移到上一个组                         |
| `Alt+PgDn`        | 将面板移到下一个组                         |
| `Alt+Home`        | 将面板移到第一个组                         |
| `Alt+End`         | 将面板移到最后一个组                       |
| `Alt+加号 (=)`    | 增加活动组宽度                             |
| `Alt+减号 (-)`    | 减小活动组宽度                             |
| `Alt+Backspace`   | 切换面板堆叠（合并/拆分）                  |
| `Alt+/`           | 打开会话菜单                               |
| `Alt+1-9`         | 按编号跳转到面板                           |
