# Панель операций

Панель операций отображает каждую фоновую файловую операцию
(копирование, перемещение, загрузку, скачивание, удаление, пакетный
перенос) и каждую фоновую команду. Каждая операция получает
собственную карточку с индикатором прогресса, источником/целью,
переданным объёмом байт и прошедшим временем.

## Открытие и закрытие

Панель открывается автоматически при старте первой фоновой операции.
Также её можно открыть из меню приложения.

Закрытие — по стандартным правилам панели (`Alt+X`, `F10` или `Esc`,
когда ничего не выбрано). С выбранной операцией `Esc` отменяет её, а
не закрывает панель — см. горячие клавиши ниже.

## Горячие клавиши

| Сочетание                       | Действие                                          |
|---------------------------------|---------------------------------------------------|
| `↑` / `↓` (или `k` / `j`)        | Перемещение курсора между операциями              |
| `Home` / `End` (или `g` / `G`)   | Переход к первой/последней операции               |
| `Space`                          | Пауза или возобновление выбранной операции        |
| `Esc` / `Delete` / `Backspace`   | Отмена выбранной операции                         |

`Esc` отменяет только когда операция выбрана; без выбора он
проваливается в стандартное закрытие панели по `Esc` на уровне
приложения.

## Контекстное меню операции

Клик по иконке типа в квадратных скобках (`[↑]`, `[↓]`, `[⧉]`, `[➜]`
…) в рамке карточки открывает popup-меню для этой операции:

- **Pause / Resume** — то же, что `Space`. Переключает состояние
  паузы.
- **Cancel** — то же, что `Esc` / `Delete`.

Когда операция на паузе, между иконкой и подписью появляется
индикатор `⏸` (например, `[↑] ⏸ Copy (upload) 23%`).

## Очистка после отмены

Отмена операции, которая уже переместила байты, **не** оставляет
мусор молча. Панель спрашивает, что делать с остатками:

### Удалённая загрузка (один файл)

Если отменяемая операция — удалённая загрузка, появляется модал:

```
Upload was cancelled

Delete partial upload 'filename'?
```

**Yes** (по умолчанию) посылает удалённую команду на удаление файла,
который был в работе. **No** оставляет его на сервере — например,
чтобы потом продолжить передачу через серверный механизм возобновления.

### Локальный/удалённый batch

Для пакетной операции, которая уже произвела файлы в целевой
директории, модал предлагает два варианта:

- **Delete copied** — удалить всё, что batch записал к этому моменту.
- **Keep copied** — оставить частичный набор как есть.

Список путей для очистки строится из отслеживаемых целей batch'а,
поэтому затрагиваются только файлы, появившиеся в результате этой
отменённой операции; ранее существовавшие файлы в целевой директории
не трогаются.
