# Файловый менеджер

Панель файлового менеджера предоставляет интуитивно понятный интерфейс для навигации по файловой системе и выполнения операций с файлами и каталогами.

## Навигация

| Комбинация        | Действие                                   |
|-------------------|--------------------------------------------|
| `↑` / `↓`         | Перемещение курсора вверх/вниз             |
| `Enter`           | Войти в каталог, просмотреть медиафайл или открыть файл |
| `Backspace`       | Перейти в родительский каталог             |
| `~`               | Перейти в домашний каталог                 |
| `PageUp` / `PageDown` | Прокрутка списка на одну страницу      |
| `Home` / `End`    | Перейти в начало/конец списка              |
| `.`               | Переключить видимость скрытых файлов       |
| `Ctrl+P`          | Переключатель каталогов                    |
| `Ctrl+G`          | Перейти по пути/URL                        |
| `Ctrl+B`          | Добавить закладку                          |
| `Tab`             | Перейти на следующую панель                |
| `Shift+Tab`       | Перейти на предыдущую панель               |

## Выделение файлов

| Комбинация        | Действие                                   |
|-------------------|--------------------------------------------|
| `Insert`          | Выделить/снять выделение с текущего файла  |
| `Shift + ↑ / ↓`   | Выделить несколько файлов подряд           |
| `Ctrl+A`          | Выделить все файлы и каталоги в панели     |
| `Escape`          | Снять все выделения                        |

## Операции с файлами

| Комбинация        | Действие                                   |
|-------------------|--------------------------------------------|
| `Ctrl+N`          | Создать новый файл                         |
| `D` / `F7`        | Создать новый каталог                      |
| `Delete` / `F8`   | Удалить выделенные файлы/каталоги          |
| `C` / `F5`        | Копировать выделенные файлы/каталоги       |
| `M` / `F6`        | Переместить/переименовать файлы/каталоги   |
| `E` / `F4`        | Открыть файл в редакторе                   |
| `R` / `F2`        | Переименовать файл/каталог                 |
| `V` / `F3`        | Просмотр файла (без запуска)               |
| `Ctrl+R`          | Обновить содержимое текущего каталога      |
| `Пробел`          | Показать информацию о файле/каталоге       |

## Поиск

| Комбинация        | Действие                                   |
|-------------------|--------------------------------------------|
| `Ctrl+F`          | Поиск файлов по имени (glob-шаблоны)       |
| `Ctrl+Shift+F`    | Поиск по содержимому файлов (regex)        |

### Поиск файлов (Ctrl+F)

Открывает модальное окно для быстрого поиска файлов по имени с использованием glob-шаблонов:
- Фильтрация файлов в реальном времени при вводе
- Результаты показывают относительные пути с цветами статуса git
- Нажмите Enter для открытия выбранного файла
- Нажмите Escape или кликните вне модала для закрытия

### Поиск по содержимому (Ctrl+Shift+F)

Открывает модальное окно для поиска текста внутри файлов с использованием регулярных выражений:
- Поиск только в текстовых файлах (бинарные файлы пропускаются)
- Большие файлы пропускаются (настраиваемый лимит в конфигурации)
- Результаты показывают путь к файлу, номер строки и контекст (3 строки)
- Найденный текст подсвечивается
- Нажмите Enter для открытия файла на найденной строке
- Нажмите Escape или кликните вне модала для закрытия

## Буфер обмена

| Комбинация        | Действие                                   |
|-------------------|--------------------------------------------|
| `Ctrl+C`          | Копировать пути выделенных элементов       |
| `Ctrl+X`          | Вырезать пути выделенных элементов         |
| `Ctrl+V`          | Вставить файлы из буфера обмена            |

## Интеграция с Git

Файловый менеджер отображает статус файлов в Git-репозиториях, подсвечивая новые, измененные и удаленные файлы.

## Просмотр медиафайлов

Файловый менеджер может просматривать изображения и видео с помощью консольных просмотрщиков.

**Логика открытия файлов:**

| Тип файла | Действие |
|-----------|----------|
| Растровые изображения (PNG, JPG, JPEG, GIF, WebP, BMP, TIFF) | ImagePanel (нативная графика) или xdg-open |
| Векторные изображения (SVG, ICO) | xdg-open (системный просмотрщик) |
| Видео (MP4, MKV, AVI, MOV, WebM, FLV, WMV, M4V) | xdg-open (системный плеер) |
| Бинарные файлы | xdg-open (системное приложение) |
| Текстовые файлы | Панель редактора |
| Исполняемые файлы | Запуск в терминале |

**Горячие клавиши:**
- `Enter` → умное открытие (см. таблицу выше)
- `F3` → просмотр файла (как Enter, но исполняемые открываются в редакторе)
- `Shift+Enter` → принудительно открыть через xdg-open (системное приложение)
- `F4` → всегда открывать в редакторе

**Нативная графика:**
termide автоматически определяет, поддерживает ли родительский терминал графические протоколы (Kitty, Sixel, iTerm2). При наличии поддержки растровые изображения рендерятся напрямую в ImagePanel без внешних инструментов.

**Поддерживаемые терминалы:**
- Kitty, WezTerm, iTerm2, Ghostty, foot - полная поддержка графики
- Другие терминалы - fallback на xdg-open

## Поддержка мыши

- **Одиночный клик**: Выбрать файл или каталог.
- **Двойной клик**: Войти в каталог или открыть файл.
- **Колесо прокрутки**: Прокрутка списка файлов.

## Отображение

### Модификаторы шрифта

- **Курсив** — символические ссылки (файлы и каталоги)
- **Жирный** — исполняемые файлы

### Порядок сортировки

Файлы отображаются группами, каждая отсортирована по алфавиту:
1. Каталоги
2. Исполняемые файлы
3. Обычные файлы

### Символические ссылки

- Модальное окно копирования содержит флажок «Создать символическую ссылку» — создаёт ссылку вместо копирования
- При переходе в каталог-ссылку файловый менеджер следует по ссылке в целевой каталог

### Примечания

- `..` (родительский каталог) не может быть выделен — используется только для навигации
- Модальные окна ввода поддерживают раскрытие `~/` в домашний каталог
