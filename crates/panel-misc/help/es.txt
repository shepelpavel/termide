  TECLAS GLOBALES

    Alt+M        Alternar menú
    Alt+H        Mostrar esta ayuda
    Alt+Q        Salir de la aplicación
    Alt+F        Abrir panel de administrador de archivos
    Alt+T        Abrir panel de terminal
    Alt+E        Abrir panel de editor
    Alt+L        Abrir panel de registro
    Alt+P        Abrir archivo de configuración en el editor
    Alt+/        Menú sesiones (nueva/cambiar/cambiar raíz)


  GESTIÓN DE PANELES

    Alt+←/→         Cambiar entre grupos de paneles (horizontal)
    Alt+↑/↓         Navegar paneles dentro del grupo (vertical)
    Alt+W/S/A/D     Navegación estilo WASD (alternativa a flechas)
    Alt+PgUp        Mover panel al grupo anterior
    Alt+PgDn        Mover panel al grupo siguiente
    Alt+Home        Mover panel al primer grupo
    Alt+End         Mover panel al último grupo
    Alt+Plus (=)    Aumentar ancho del grupo activo
    Alt+Minus (-)   Disminuir ancho del grupo activo
    Alt+X / Esc     Cerrar panel actual
    Alt+Delete      Cerrar panel actual
    Alt+Backspace   Alternar apilamiento de paneles (combinar/separar)


  TECLAS DEL ADMINISTRADOR DE ARCHIVOS

    ↑↓           Navegar lista de archivos
    Enter        Abrir archivo o directorio (ejecutar ejecutables)
    F3           Ver archivo (ejecutables en el editor)
    Shift+Enter  Abrir con aplicación del sistema (xdg-open)
    Backspace    Ir al directorio padre
    Home         Saltar al primer elemento
    End          Saltar al último elemento
    PgUp / PgDn  Desplazar página arriba/abajo
    Insert       Seleccionar/deseleccionar elemento
    Space        Mostrar información del archivo
    Ctrl+A       Seleccionar todos los elementos
    F4           Abrir archivo en el editor
    Ctrl+N       Crear nuevo archivo
    Ctrl+F       Buscar archivos
    d / D / F7   Crear nuevo directorio
    c / C / F5   Copiar elementos seleccionados
    m / M / F6   Mover elementos seleccionados
    F8 / Delete  Eliminar elementos seleccionados
    Ctrl+C       Copiar al portapapeles
    Ctrl+X       Cortar al portapapeles
    Ctrl+V       Pegar desde el portapapeles


  TECLAS DEL EDITOR

    Ctrl+S / F2  Guardar archivo
    Ctrl+C       Copiar al portapapeles del sistema
    Ctrl+V       Pegar desde el portapapeles del sistema
    Ctrl+X       Cortar al portapapeles del sistema
    Ctrl+Y       Rehacer
    Ctrl+Z       Deshacer
    Ctrl+F       Buscar texto (modal interactivo)
    F3 / Shift+F3 Navegar coincidencias de búsqueda
    Tab / Shift+Tab Navegar coincidencias (cuando la búsqueda está activa)
    Ctrl+H       Reemplazar texto (modal interactivo)
    Ctrl+G       Ir a línea
    Escape       Cerrar búsqueda/modal, o cerrar panel


  TECLAS DEL PANEL GIT STATUS

    Tab          Cambiar foco (selectores / archivos / botones)
    ↑↓           Navegar archivos o elementos de lista
    Space        Preparar/despreparar archivo
    Insert       Seleccionar archivo (operaciones por lotes)
    Delete       Quitar del área de preparación
    Ctrl+S       Preparar archivos seleccionados
    Ctrl+U       Despreparar archivos seleccionados
    Ctrl+R       Actualizar estado
    Enter        Ejecutar botón o seleccionar elemento


  TECLAS DEL PANEL GIT LOG

    Tab          Cambiar foco (ramas / commits)
    ↑↓ / j/k     Navegar commits
    PgUp / PgDn  Página arriba/abajo
    Home / g     Ir al primer commit
    End / G      Ir al último commit
    Enter / d    Mostrar detalles del commit
    c / C        Copiar hash del commit
    Ctrl+R       Actualizar registro


  Use las teclas de flecha para desplazar este texto de ayuda.
