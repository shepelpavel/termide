  ปุ่มกด GLOBAL

    Alt+M        สลับเมนู
    Alt+H        แสดงความช่วยเหลือนี้
    Alt+Q        ออกจากแอปพลิเคชัน
    Alt+F        เปิดแผงจัดการไฟล์
    Alt+T        เปิดแผงเทอร์มินัล
    Alt+E        เปิดแผงเอดิเตอร์
    Alt+L        เปิดแผงบันทึก
    Alt+P        เปิดไฟล์คอนฟิกในเอดิเตอร์
    Alt+/        เมนูเซสชัน (ใหม่/สลับ/เปลี่ยนราก)
    Alt+G        เปิดแผง Git Status


  การจัดการแผง

    Alt+←/→         สลับระหว่างกลุ่มแผง (แนวนอน)
    Alt+↑/↓         นำทางแผงภายในกลุ่ม (แนวตั้ง)
    Alt+W/S/A/D     นำทางแผงแบบ WASD (ทางเลือกของลูกศร)
    Alt+PgUp        ย้ายแผงไปยังกลุ่มก่อนหน้า
    Alt+PgDn        ย้ายแผงไปยังกลุ่มถัดไป
    Alt+Home        ย้ายแผงไปยังกลุ่มแรก
    Alt+End         ย้ายแผงไปยังกลุ่มสุดท้าย
    Alt+Plus (=)    เพิ่มความกว้างของกลุ่มที่ใช้งาน
    Alt+Minus (-)   ลดความกว้างของกลุ่มที่ใช้งาน
    Alt+X / Esc     ปิดแผงปัจจุบัน
    Alt+Delete      ปิดแผงปัจจุบัน
    Alt+Backspace   สลับการซ้อนแผง (รวม/แยก)


  ปุ่มกดจัดการไฟล์

    ↑↓           นำทางรายการไฟล์
    Enter        เปิดไฟล์หรือไดเรกทอรี (รันไฟล์ที่สั่งการได้)
    F3           ดูไฟล์ (ไฟล์ที่สั่งการได้เปิดในเอดิเตอร์)
    Shift+Enter  เปิดด้วยแอปพลิเคชันระบบ (xdg-open)
    Backspace    ไปยังไดเรกทอรีหลัก
    Home         ข้ามไปยังรายการแรก
    End          ข้ามไปยังรายการสุดท้าย
    PgUp / PgDn  เลื่อนหน้าขึ้น/ลง
    Insert       เลือก/ยกเลิกการเลือกรายการ
    Space        แสดงข้อมูลไฟล์
    Ctrl+A       เลือกรายการทั้งหมด
    F4           เปิดไฟล์ในเอดิเตอร์
    Ctrl+N       สร้างไฟล์ใหม่
    Ctrl+F       ค้นหาไฟล์
    d / D / F7   สร้างไดเรกทอรีใหม่
    c / C / F5   คัดลอกรายการที่เลือก
    m / M / F6   ย้ายรายการที่เลือก
    F8 / Delete  ลบรายการที่เลือก
    Ctrl+C       คัดลอกไปยังคลิปบอร์ด
    Ctrl+X       ตัดไปยังคลิปบอร์ด
    Ctrl+V       วางจากคลิปบอร์ด


  ปุ่มกดเอดิเตอร์

    Ctrl+S / F2  บันทึกไฟล์
    Ctrl+C       คัดลอกไปยังคลิปบอร์ดของระบบ
    Ctrl+V       วางจากคลิปบอร์ดของระบบ
    Ctrl+X       ตัดไปยังคลิปบอร์ดของระบบ
    Ctrl+Y       ทำซ้ำ
    Ctrl+Z       ยกเลิก
    Ctrl+F       ค้นหาข้อความ (โมดัลแบบโต้ตอบ)
    F3 / Shift+F3 นำทางผลการค้นหา
    Tab / Shift+Tab นำทางผลลัพธ์ (เมื่อการค้นหาทำงาน)
    Ctrl+H       แทนที่ข้อความ (โมดัลแบบโต้ตอบ)
    Ctrl+G       ไปยังบรรทัด
    Escape       ปิดการค้นหา/โมดัล หรือปิดแผง


  ปุ่มกดแผง GIT STATUS

    Tab          สลับโฟกัส (ตัวเลือก / ไฟล์ / ปุ่ม)
    ↑↓           นำทางไฟล์หรือรายการ
    Space        สเตจ/อันสเตจไฟล์
    Insert       เลือกไฟล์ (การดำเนินการแบบกลุ่ม)
    Delete       ลบจากสเตจ
    Ctrl+S       สเตจไฟล์ที่เลือก
    Ctrl+U       อันสเตจไฟล์ที่เลือก
    Ctrl+R       รีเฟรชสถานะ
    Enter        ดำเนินการปุ่มหรือเลือกรายการ


  ปุ่มกดแผง GIT LOG

    Tab          สลับโฟกัส (สาขา / คอมมิต)
    ↑↓ / j/k     นำทางคอมมิต
    PgUp / PgDn  หน้าขึ้น/ลง
    Home / g     ไปยังคอมมิตแรก
    End / G      ไปยังคอมมิตสุดท้าย
    Enter / d    แสดงรายละเอียดคอมมิต
    c / C        คัดลอกแฮชคอมมิต
    Ctrl+R       รีเฟรชบันทึก


  ใช้ปุ่มลูกศรเพื่อเลื่อนข้อความความช่วยเหลือนี้
